# LeadGenius AI 🚀

LeadGenius AI is an advanced open-source Lead Intelligence & Sales Strategy Dashboard powered by Google's Gemini AI models.

## Features
- 🤖 **AI-Powered Business Profiles**: Auto-fill your business profile using Magic Fill.
- 📊 **Sales Strategy Generation**: Get comprehensive sales strategies tailored to your business.
- 🎯 **Lead Generation**: Discover highly qualified leads using Google Maps & Search grounding.
- ✉️ **Outreach Templates**: Generate personalized outreach messages (Email, LinkedIn, Cold Call).
- 📋 **Export**: Export your generated leads to CSV.

## Tech Stack
- Next.js 15 (App Router)
- React
- Tailwind CSS
- Google Gemini API (`@google/genai`)
- Lucide React (Icons)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/leadgenius-ai.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Copy `.env.example` to `.env.local` and add your Gemini API key.
   ```bash
   cp .env.example .env.local
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.
